### 功能

* 只调用沪江小D查询日语，可以点击语音按钮发音
* 默认是日中查询，没有结果时自动使用中日查询，日中查询有结果有添加主动中日查询的按钮
* 有多个意思和发音时，可以点击切换
* 支持浏览器：Google Chrome、Firefox、Safari、Firefox for Android（无拖动）
* 支持图标拖动
* 支持面板自动调整位置
